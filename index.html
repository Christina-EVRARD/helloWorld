Quête Github

Git est un système de contrôle de version Git t'aide à conserver l'historique de tes fichiers et à fournir des outils pour travailler en équipe sur les mêmes fichiers de manière concurrente Github est une plateforme en ligne qui propose des services tels que l'hébergement de code mais c'est aussi un lieu social où tu peux participer à des projets open source, partager et découvrir de nouveaux projets.
